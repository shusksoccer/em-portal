# 転写（Transcript / Jefferson記法）

録音・録画された相互行為のデータを、詳細に分析するために文字や記号で書き起こす作業、およびその記述ルールのことです。会話分析の創始者の一人であるゲイル・ジェファーソンが開発した「ジェファーソン記法」が世界標準として用いられています。

## 定義

会話を紙の上で精密に再現するために、話された言葉だけでなく、声の高さ、間の長さ（秒単位）、息づかい、言葉の重なりまでを特定の記号を使って書き表す手法のことです。単なる「議事録」ではなく、会話という現象を科学的に記述するための精密な「楽譜」のようなものを指します。

## なぜこれが重要か

社会の秩序は、言葉の「内容」だけでなく、その「言い方」や「タイミング」といった極めてミクロな細部に宿っているからです。心理的な解釈（例：「怒っているように見えた」）を排除し、0.1秒単位の沈黙や声の伸びを客観的に記述することで、参加者がその細部をいかに利用して社会的なメッセージを送受信しているかを分析可能にします。

## 身近な具体例

**例1：授業中の「間」の分析**
先生が「この問題わかる人？」と問いかけた後の `(2.5)` という2.5秒の沈黙を記述する状況。この長い沈黙は、単なる空白ではなく、クラス全員が「誰も手を挙げようとしていない」という気まずい状況を共有していることを示すデータとなります。

**例2：友達同士の盛り上がり**
生徒A「マジで[うけるんだけど]」と言いながら生徒B「[それな！]」と重なって返す状況。`[ ]` で囲まれた部分が同時に発話されていることを示しており、この重なり方自体が「共感・盛り上がり」の達成として分析できます。

## 主要な転記記号（ジェファーソン記法・最小セット）

| 記号 | 意味 | 使用例 |
|---|---|---|
| `(0.0)` | 秒単位の沈黙 | `(1.2)` = 1.2秒の間 |
| `(.)` | マイクロポーズ（0.2秒未満） | `あ(.)あのさ` |
| `[ ]` | 発話の重なり（同時発話） | `[そうだよ]` で挟む |
| `=` | ラッチング（隙間なしの継続） | `はい=そうです` |
| `:` | 音の引き伸ばし | `あのさ:::` |
| `-` | 言いかけの中断 | `そうじゃな-` |
| `(( ))` | 分析者による注釈（非言語情報） | `((頷く))` |

## よくある誤解

「時間がかかるし、そこまで細かく書く必要はない」と思われがちですが、会話分析では「転記をすること自体が分析の核心であり、何度もスロー再生して細部を書き写すことで初めて、肉眼や普通の耳では気づかなかった重要な社会的な手続きが見えてくる」と考えます。

## この概念で見ると分かること

一見ランダムに見える「あー」や「えーと」といった言葉の淀みが、実は「次の言葉を探している」とか「相手の意見に反対しにくいので言葉を濁している」といった、高度な社会的な調整を行っていることが観察可能になります。
