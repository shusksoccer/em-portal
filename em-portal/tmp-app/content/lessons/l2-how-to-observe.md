## 導入：秩序はどこで「見える」ようになるか

L1で学んだ「説明可能性」「文脈依存性」は、どうすれば実際のデータで確認できるでしょうか。EMでは、観察者自身が記録をとることで、その場の秩序の達成を後から検証可能にします。

:::example 高速道路のドライバーたち（第3章の知見から）
高速道路では、ドライバーたちは事前に誰と調整の約束をするわけでなく、お互いの速度・車間距離を調整し、視線やウインカーを通じて協調しています。全体として「スムーズな交通」という秩序が形成されていますが、それは誰かが「命令」したわけではなく、参加者全員の行為の積み重ねで達成されています。

この秩序を記録するには、個々のドライバーの行為（ハンドル操作・ウインカー・速度変化）を時系列で記録する必要があります。
:::

## 秩序の達成とは

:::concept 秩序はその場で「作られる」
EMの視点では、秩序とは外部から与えられる固定ルールではなく、参加者がお互いの振る舞いを調整し、協調し合うことで**そのつど達成されるもの**です。

観察の目的は、この「達成のプロセス」を記録することです。「秩序があった」という結論だけでなく、「誰が・いつ・何をすることで秩序が成立したか」を記録します。
:::

:::example 行列（Queue）の道徳的秩序
コンビニのレジ待ちを考えてください。列に並ぶという行為は、参加者が「誰が先か」という順番を共有し、割り込みを互いに監視し合うことで成り立っています。ルール書はありませんが、割り込みがあると「非常識だ」という反応が起きます。

この観察ログには何が必要でしょうか？
- 各人物の位置と移動方向
- 身体の向き・視線の方向
- 他者への発話（あれば）
- 割り込みへの反応（あれば）

観察者はこれらを時系列で記録します。
:::

## 観察ログの書き方

:::concept 観察ログの基本単位：4列
観察ログは「時刻 / 発話 / 行為 / 注記」で記録します。

- **時刻**: 順序を固定する（秒単位が望ましい）
- **発話**: 実際に言われた言葉をそのまま書く（要約しない）
- **行為**: 非言語的な動作（ジェスチャー・移動・視線など）
- **注記**: 最小限の補足（解釈は書かない）

事実と解釈を分けることが、後の分析精度を上げます。
:::

:::transcript 廊下でのやりとり（ログ例）
時刻 / 発話 / 行為 / 注記
12:05 / A「プリント、今出す？」/ Aが鞄を触る /
12:05 / — / Bが鞄を開ける / 発話なしで応答
12:06 / C「先生まだ来てないよ」/ Cが廊下を見る /
12:06 / A「じゃあ先に並ぼう」/ Aが立ち上がる /
---
2行目Bの行為は発話なしで「今出す意向」を示している。3行目Cの補足で場面条件が共有され、4行目で次行動の提案が出る。この一連が「次の授業準備」という秩序を共同で達成している。
:::

## 観察でよくある誤り

:::caution 解釈語を事実欄に書いてしまう
**NG（解釈語）：** 「雰囲気が重かった」「Aはやる気なさそう」
→ 何が観察されたのかが分からない

**OK（事実記述）：** 「A・Bともに10秒間発話なし、二人とも下を向いていた」
→ 読み手が場面を確認できる

抽象的な語を使う場合は、**必ず対応する具体的な行為・発話をセットで示してください**。
:::

:::caution 「場所取り」のような空間の秩序も記録対象
EM研究では、物理的な空間の配置も秩序の一部として扱います。例えば「机の上に荷物が置いてある＝その席は使用中」という理解は、参加者の共有された知識です。このような「物の配置が示す意味」も観察ログの対象になります。
:::

## 倫理確認

:::caution 観察前に必ず確認する
- 個人が特定できる情報を記録しない
- 観察対象に不利益が及ぶ内容を扱わない
- 録音・録画する場合は事前に同意を得る
- データを授業外で共有しない
:::

## やってみよう

:::exercise
1. 1〜2分の身近な場面を選び、4列ログを最低3行作る。
2. ペアで交換して、「解釈が入っている」行を1か所指摘する。
3. 指摘を受けて修正し、次の観察で意識するルールを3つ書く。
:::
