## このワークの目的

当たり前の行為が、参加者の方法で秩序化されていることを、**評価せずに「順番（順序性）」で記述する**。

:::example この観察が見せてくれる驚き
コンビニで会計が終わり、客が背を向けて歩き出した後でも「ありがとうございました」と声をかけられる。相手はもう聞いていないのに、なぜ？　それはお礼ではなく、**取引の終了を宣言する順番**だからです。言葉の「意味」ではなく「順番の中での機能」を見ると、日常の何でもない場面が精密な設計図をもっていることに気づきます。このワークはその設計図を書き出す練習です。
:::

## 手順（生徒向けステップバイステップ）

**ステップ1**　班で「観察場面」を1つ選ぶ（人ではなく場面を選ぶ）。

> 観察してよい場面の例：授業の開始、挙手→指名、プリント配布、席の移動、昼休みの並び、など。

**ステップ2**　ルールを3つ確認してから始める。
- 個人名を書かない（A・B・Cなどで記録）
- 「良い・悪い」を書かない（評価しない）
- 起きた順番を書く

**ステップ3**　3〜5分観察し、起きたことを矢印で順番に書く。

```
例：A挙手 → 教師がAを指名 → Aが立って話す → 教師「ありがとう」 → Aが着席
```

**ステップ4**　「次に起きるはず」（背景期待）を2つ書く。

```
例：呼びかける → 返事が返ってくるはず
    質問する  → 誰かが答えるはず
```

**ステップ5**　予想と違う瞬間（沈黙・遅れ・言い直し・別の人が入るなど）を見つけたら ★ を付ける。

**ステップ6**　★の後に起きた「立て直し」を書く。

```
例：沈黙5秒 → 教師が別の生徒を指名する → 再開
```

**ステップ7**　班の結論を1文でまとめる：

> 「この場面では、（　　　）という期待があり、崩れると（　　　）が起きやすい。」

**ステップ8**　30秒で共有：結論文と根拠（どの出来事か）を言う。

```
①　→　②　→　③　→　④　→　⑤
```

### 背景期待（「次に起きるはず」）
- 期待①：
- 期待②：

### ★ 予想と違った瞬間
- 何が起きたか：
- その後の立て直し：

### 結論1文
> この場面では、（　　　）があり、崩れると（　　　）が起きやすい。

## 評価基準（ルーブリック）

| 観点 | 3（できる） | 2（だいたい） | 1（これから） |
|---|---|---|---|
| 順番の具体性 | 出来事が順序で追える | 一部曖昧だが追える | 評価語が多く順番が不明 |
| 期待と根拠 | 「次に起きるはず」と根拠がある | 期待はあるが根拠が弱い | 期待が書けない／一般論 |
| 倫理・配慮 | 匿名化・非揶揄が徹底できている | 概ね守れている | 個人特定・揶揄が混入 |
